Staffing Evidence
Overview

Staffing Evidence is a web application designed to collect, analyse, and report hospital staffing data over time. Its primary purpose is to provide objective evidence of staffing levels, patient acuity, and workload pressures to support workforce planning, quality improvement, and advocacy.

The application enables healthcare staff to record staffing information for each shift, review historical data, generate reports, and identify trends through charts and analytics.

V1 Features
Record staffing levels for each shift
Record planned vs actual staffing
Record skill mix (RN, EN, AIN, NUM, etc.)
Record permanent, casual, and agency staff
Record patient numbers, high acuity, and high dependency patients
Record operational metrics (e.g. falls, rapid responses, Code Blue)
Search and filter historical shift data
Generate charts and dashboards
Export data to CSV

Planned V2
